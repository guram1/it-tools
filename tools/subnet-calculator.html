How to Use the Subnet Calculator

Enter an IP address and subnet mask (in CIDR notation like /24 or dotted decimal like 255.255.255.0) to calculate all subnet details including network address, broadcast address, and usable host range.

Understanding Subnet Results

  • Network Address — The first address in the subnet, identifies the network itself
  • Broadcast Address — The last address in the subnet, used to send packets to all hosts
  • First/Last Usable Host — The range of IP addresses you can assign to devices
  • Wildcard Mask — Inverse of subnet mask, used in Cisco ACLs and OSPF

Common Subnet Masks

  • /8 (255.0.0.0) — Class A, 16 million hosts
  • /16 (255.255.0.0) — Class B, 65,534 hosts
  • /24 (255.255.255.0) — Class C, 254 hosts
  • /30 (255.255.255.252) — Point-to-point links, 2 hosts
  • /32 (255.255.255.255) — Host route, 1 host

Private IP Address Ranges (RFC 1918)

The calculator identifies private vs public IP addresses:

  • 10.0.0.0/8 — 10.0.0.0 to 10.255.255.255
  • 172.16.0.0/12 — 172.16.0.0 to 172.31.255.255
  • 192.168.0.0/16 — 192.168.0.0 to 192.168.255.255
